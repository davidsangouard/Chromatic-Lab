# Chromatic Lab

Générateur de palettes de couleurs harmonieuses et de patterns CSS minimaliste.

## Fonctionnalités

- Génération de palettes basées sur la théorie des couleurs
- Harmonies: analogues, complémentaires, triadiques, tétradiques, monochromatiques
- Patterns CSS personnalisables
- Export des palettes en variables CSS
- Interface élégante et minimaliste

## Harmonies disponibles

- **Monochromatiques**: Variations d'une seule teinte
- **Analogues**: Couleurs adjacentes sur le cercle chromatique
- **Complémentaires**: Couleurs opposées
- **Triadiques**: Trois couleurs équidistantes
- **Tétradiques**: Quatre couleurs en rectangle
